เว็บนี้ช่วยตรวจอะไร

Research Logic

ตรวจว่า problem, objective, method และ result ต่อกันจริงหรือไม่

References

ตรวจ reference quality, missing citations, uncited entries, abstract fields และ citation ที่ไม่ support claim

Evidence

ตรวจ dataset, baseline, metric, raw result, script และ error analysis ที่รองรับ claim

Ethics / IRB / Legal

ช่วย flag human subjects, personal data, consent, scraping, dataset license, copyright, ToS และ NDA

AI Use

ตรวจว่า AI อาจทำให้เกิด claim/citation/result ลอยหรือไม่ และผู้เขียนตรวจจากแหล่งจริงแล้วหรือยัง

Writing Hygiene

ตรวจ typo, ชื่อย่อ, การปนไทย/อังกฤษ, forward references และ generic AI writing

Privacy และข้อจำกัด

เว็บนี้ไม่รับข้อมูลวิจัยจากผู้ใช้

หน้า HTML ชุดนี้เป็น static prompt list ไม่มีฟอร์มรับข้อมูล ไม่มีระบบอัปโหลด และไม่ได้เก็บข้อมูลงานวิจัยของผู้ใช้ไว้บน server ใด ๆ

AI เป็นผู้ช่วยตรวจ ไม่ใช่เจ้าของงาน ไม่ใช่อาจารย์ที่ปรึกษา ไม่ใช่ reviewer จริง และไม่ใช่ผู้พิจารณา IRB/legal compliance ผู้เขียนต้องตรวจทุก claim จากแหล่งจริงเอง

ห้ามให้ AI แต่งข้อมูลเกินหลักฐาน เช่น citation, DOI, venue ranking, metric, result, IRB/legal requirement หรือข้อกฎหมายเอง

Credits

จัดทำโดย พันธุ์ปิติ เปี่ยมสง่า
