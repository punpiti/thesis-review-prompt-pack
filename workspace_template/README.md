# Thesis Review Workspace Template

คัดลอกโฟลเดอร์นี้ไปเป็น workspace ใหม่ของงานที่ต้องการตรวจ แล้ววางไฟล์ thesis/manuscript และ reference ตามโครงนี้

```text
my_thesis_review/
  AGENTS.md
  CLAUDE.md
  00_context_notes.md
  input/
  01_working_text/
  02_references/
    bib/
    pdf/
    extracted_md/
    official_pages/
  03_results/
  04_review_notes/
  prompt_pack/
```

หลังจัดไฟล์แล้ว เปิด workspace นี้ด้วย Codex / Claude Code / Antigravity แล้วสั่ง:

```text
ให้ประเมิน
```

AI ควรสร้าง review notes ใน `04_review_notes/` ก่อน ห้ามแก้ thesis ต้นฉบับทันที
