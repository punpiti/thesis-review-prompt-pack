# 00_inbox

ถ้ายังไม่รู้ว่าควรวางไฟล์ไว้ตรงไหน ให้ใส่ไฟล์รวมไว้ในโฟลเดอร์นี้ก่อน

ตัวอย่าง:

- thesis/manuscript `.tex`, `.docx`, `.pdf`, `.md`, `.txt`
- รูป ตาราง ผลทดลอง log script
- BibTeX หรือ reference list
- PDF reference
- DOI page, publisher page, official abstract

จากนั้นเปิด workspace ด้วย Codex หรือ Claude Code แล้วสั่ง:

```text
init workspace จากไฟล์รวม
```

AI จะจัดไฟล์เข้าโฟลเดอร์ที่เหมาะสม และควรคงไฟล์ต้นฉบับใน `00_inbox/` ไว้ก่อน ไม่ลบไฟล์เอง
