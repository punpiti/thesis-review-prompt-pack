# 02_references/pdf

ใส่ PDF ของ reference ต้นฉบับที่อ้างใน thesis/manuscript

AI ต้องใช้ PDF เหล่านี้เป็นหลักฐาน ห้ามแต่ง metadata, abstract, DOI, venue หรือ year เอง

เมื่อ AI อ่าน PDF ใด ให้บันทึกผลอ่านเป็น Markdown หนึ่งไฟล์ต่อหนึ่ง PDF ใน `../extracted_md/` ด้วย ก่อนนำข้อมูลไปเติม `references.bib`
